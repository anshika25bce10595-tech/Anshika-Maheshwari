# Academic tracker
#Problem statement
Students often forget assignments, miss deadlines, and struggle to track their grades. A simple tool is needed to organize tasks, check due dates, and monitor academic performance easily.
#Scope of the project
This project is a simple, student-friendly grade and assignment tracker created using Python in a Jupyter Notebook.Its main goal is to help students stay organized by keeping all their assignments, due dates, and grades in one place.The scope is limited to console-style interaction inside the notebook,without any advanced UI or external databases.
#Target users
The primary users of this project are students who need a simple way to track assignments, due dates, and grades. It is especially useful for learners who prefer an organized system to manage their academic tasks and monitor their performance.
#High-level feature
Lets students record assignments with due dates and grades.
Automatically calculates GPA and overall performance.
Shows pending, completed, and overdue tasks clearly.
Saves all data so progress is not lost between sessions.
Runs smoothly inside a Jupyter Notebook with simple menu-based interaction.
